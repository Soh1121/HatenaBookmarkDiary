# 2020年3月12日のはてなブックマークテクノロジーメモ

## 1. [「入門GUI」が発売されました](https://hashrock.hatenablog.com/entry/2020/03/11/131023)

### 1.1 概要

- GUIコンポーネントをいちから書いてみようという内容の本を発売
- 最後に座談会パートを設けて好評
- タッチデバイスの普及もあってユーザー要求に直接操作UIが含まれることが増えてきていて、大変な実装に怯んでいられなくなった

### 1.2 感想

- GUIは難しくて手が出せないでいるが、いつかは触ってみたい
- こういう本を覚えておきたい
- GUIを思うままに作れたら楽しいだろうな…

## 2. [Vue.jsを使用してさまざまなフォームを簡単に実装できる -Vue Formulate](https://coliss.com/articles/build-websites/operation/javascript/build-forms-with-vue.html)

### 2.1 概要

- よく使われるフォームをVue.jsで簡単に実装できるVue Formulateの紹介
- サインアップ・ログイン・メールアドレス・フォイルのアップロード・アンケートなどさまざまなフォームをサポート
- ドキュメントや入力要素・タイプごとに実装コードも紹介

### 2.2 感想

- Vueを使ったフォームの実装方法の紹介ということで、個人開発に使えそうな情報が手に入ってよかった
- 例も見た感じわかりやすそうなので、触ってみたい
- 自分で作らなくてはというイメージのものも、もしかしたら簡単に作る方法があるかもしれないのて意識して調べるようにしたい
